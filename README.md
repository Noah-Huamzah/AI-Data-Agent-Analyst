# AI-Data-Agent-Analyst
Built an LLM-driven AI system to convert natural language into optimized SQL for Oracle Database. Developed scalable FastAPIs with session memory, query history, and async processing. Implemented auto schema-aware query generation using embeddings and optimized performance via caching, connection pooling, and fault-tolerant design.
